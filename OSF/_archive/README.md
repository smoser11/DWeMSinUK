# OSF `_archive/` — retired materials

Files under this directory are **not** part of the current SIR submission
replication package. They are kept here for revision history and to make it
easy for anyone who wants to explore the retired pathways.

Nothing under `OSF/_archive/` is referenced by `00-main_pipeline.R`, by the
submitted manuscript (`OSF/manuscript/manuscript.qmd`), or by the online
appendix book (`OSF/manuscript/appendix/`).

## Contents

### `R_scripts_unused/`

Analysis scripts that were previously part of the R pipeline but produce
outputs not cited in the submitted paper or appendix.

| Script | Why retired |
|---|---|
| `08-netclust_subgroup_analysis.R` | Subgroup analysis by nationality using `netclust`. Not cited in the paper; `netclust` is not on CRAN and requires a source build on R ≥ 4.5. |
| `08b-simple_subgroup_analysis.R` | Simpler variant of the same subgroup analysis. Not cited in the paper. |
| `09-create_latex_tables.R` | Produces standalone LaTeX table files. The current paper renders tables directly via Quarto, so this is redundant. |

### `output_tables_unused/`

Output CSV files that no script consumes and no paper element cites.

| File | Notes |
|---|---|
| `AppendixA6_estimate_ranges.csv` | Old sensitivity summary; superseded by AppendixA1–A5. |
| `Table6_NSUM_main_results.csv` | Duplicate/precursor of `rds_nsum_comparison.csv` (which is Table 6's actual source). |

### `output_figures_unused/`

Figure PNGs generated by earlier pipeline runs but not cited by the paper or appendix.

| File(s) | Notes |
|---|---|
| `convergence_access_to_help_rds.png` (+ 6 others) | Bayesian MCMC convergence diagnostics; useful for internal validation but not published. |
| `esm_b1_rds_methods_comparison.png` | Earlier draft of Figure D.2; superseded by `figA2_rds_comparison.png` in `OSF/manuscript/figures/`. |
| `figure_nsum_moderate.png` | Earlier draft of Figure D.3; superseded by `figA3_nsum_forest.png` in `OSF/manuscript/figures/`. |

### `supporting_ESMs_retired/`

Original four `ESM_*.docx` files whose content has been folded into the online
appendix book (`OSF/manuscript/appendix/`). See `supporting_ESMs_retired/README.md`
in that subfolder for the ESM → appendix-section mapping.

### `appendix-v5-retired.qmd`

Single-file v5-era Quarto source for the appendix; superseded by the four-file
Quarto Book in `OSF/manuscript/appendix/`.

## Restoring

To bring an archived script back into the active pipeline: move it back into
`OSF/R/analysis/` and add the corresponding `run_step()` call to
`OSF/R/00-main_pipeline.R`.
